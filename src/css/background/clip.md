> css3

## clip
`배경색`에 대한 범위를 지정합니다.

### border-box
기본값.
외각선 영역까지 배경색를 배치 합니다.

```css
background-clip:border-box;
```

### padding-box
배경색이 선을 제외한 패딩 영역까지 배치를 합니다.

```css
background-clip:padding-box;
```

### content-box
컨덴츠 영역에만 배경색이 배치 됩니다.

```css
background-clip:content-box;
```
