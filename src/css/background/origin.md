# origin과 clip

## origin
배경 이미지가 채워지는 영역을 지정합니다.

### padding-box
백그라운드 이미지가 요소의 `패딩 영역`에도 포함되어 배치를 합니다.

```css
background-origin:padding-box;
```

### border-box
외각선 영역까지 배경 이미지를 배치 합니다.

```css
background-origin:border-box;
```

### content-box
컨덴츠 영역에만 배경이미지가 배치가 됩니다.

```css
background-origin:content-box;
```
