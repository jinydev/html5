# HTML 사이트만들기 연습
---
HTML, CSS를 이용하여 사이트를 만들어 보는 따라하기 실습을 연습해 봅니다.

<br>

## 스타일 초기화
---
크로스 브라우저에서 공통적인 디자인을 적용하기 위해서 스타일 시트를 초기화 합니다.

```css
* {
    margin: 0;
    padding:0;
}

body {
    font-family: sans-serif;
}

li {
    list-style: none;
}

a {
    text-decoration: none;
}

img {
    border: 0;
}
```
