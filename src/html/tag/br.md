# 문단 분리

br 테그는 줄바꿈의 breaks를 의미합니다.

```html
<br>
```

```html
<br/>
```



## 문단과 줄바꿈
문단과 문단은 일정간격이 존재 합니다.

같은 문단 내에서 줄바꿈을 할때 br 테그를 사용합니다.


## br
---
html에서는 엔터 다음줄을 브라우저에 직접 표현을 하지 않습니다.
엔터와 같은 다음줄을 출력하기 위해서는 `<br>`테그를 사용해야 합니다.
