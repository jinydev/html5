# 본문 테그
---
html의 본문을 작성할때는 어떤 테그를 사용할까요? 본문을 작성을 할때에는 `p` 테그를 사용합니다.


## 단락
---
p 테그는 영어의 단락(paragraph)의 앞글자를 의미 합니다. p 테그는 본문의 단락을 지정할 때 사용을 합니다.

```html
<p>본문 내용을 작성합니다.</p>
```

## 실습 예제
---
문단을 작성해 보도록 합니다.

```html
<!DOCTYPE html>
<html>
    <head> 
    </head>
    <body>
        
        <h1>안녕하세요</h1>
        
        <p>
            날씨가 참 좋네요. 오늘도 좋은 하루가 되었으면 합니다.
        </p>

    </body>
</html>
```

<br>

## 문단간격
---
문단과 문단은 간격을 가지게 됩니다. 문단은 `p테그`로 감싸서 생성을 하게 됩니다.
