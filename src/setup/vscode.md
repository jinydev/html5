#  편집기
HTML및 코드를 작성하기 위해서 코드 편집기를 설치 합니다. 코드를 편집할 수 있는 도구는 많이 존재합니다.

* vscode
* 서브라임 텍스트
* 아톰 에디터
* 브라켓

## VSCode
VScode는 마이크로소프트의 무료로 배포하는 코드 편집기 입니다.
https://code.visualstudio.com/


## 설치방법
VSCode 설치 및 활용법은 [vscode.jiny.dev](https://vscode.jiny.dev)를 참조하세요.
