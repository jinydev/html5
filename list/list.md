# 목록테그

## 목록의 종류
순가 있는 목록, 순서가 없는 목록으로 구분됩니다.

리스트의 종류를 나타네는 테그와 요소를 나타내는 테그의 합으로 구성을 합니다.


## 리스트 요소
리스트의 요소는 li 테그를 이용하여 생성합니다.

## ol
ordered list의 약자 입니다.
순서가 있는 목록을 생성합니다.

```html
<ol>
    <li></li>
    <li></li>
</ol>
```

순서에 대한 번호가 부여가 됩니다.


## ul
unordered list의 약자 입니다.
순서가 없는 목록을 생성합니다.


```html
<ul>
    <li></li>
    <li></li>
</ul>
```

순서가 없는 테그는 요소 앞에 `점`이 표시됩니다.


## 정의목록

* dl : 정의 목록
* dt : 정의 용어
* dd : 정의 설명
