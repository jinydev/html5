속성설정

html 문서의 언어 속성은 html 테그 안에 삽입을 합니다.

<html lang="ko">
중간생략...
</html>

lang속성은 문서의 언어 타입을 지정합니다.

언어의 종류

속성 설정값은 다음과 같습니다.
