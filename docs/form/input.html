type 형식 속성

input은 다양한 속성값을 부여하여 처리를 할 수 있습니다.

text

<input type="text">

submit

입력롬을 서버로 전송을 위한 버튼을 출력합니다.

<input type="submit">

checkbox

file

hidden

image

password
