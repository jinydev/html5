글자의 스타일을 지정할 수 있습니다.

html에서 다야한 글자의 모양을 출력할 수 있습니다.

굵은 표시

강조하고 싶은 글자를 굵은(bold) 처리를 할 수 있습니다.

<b>중요</b>합니다.

이탤릭체

기울어진 글자를 표시합니다.

<i> </i>

i테그는 최근 사용 빈도가 줄어들고 있습니다.
이를 대체하여 아이콘을 출력할때 i를 속성과 같이 사용하는 경우가 많습니다.

작은글자

작은 글자를 출력합니다.

<small> </small>

위 첨글자

위쪽에 붙는 글자로 출력이 됩니다

<sub> </sub>

밑줄

밑에 붙는 글자로 출력이 됩니다.

<ins> </ins>

삭제표시

가운데 줄이 그어진 글자를 출력합니다.

<del> </del>

CSS 응용

본 글자의 모양은 HTML 테그를 이용하여 사용을 할 수도 있지만, CSS를 이용하여 처리도 가능합니다.
