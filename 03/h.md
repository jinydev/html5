# 제목 테그
html은 제목을 쉽게 표기하여 위하여 미리 사전에 정의한 테그들이 존재합니다.

## 글자 크기
html의 제목 테그는 `<h숫자> </h숫자>` 형태로 사용합니다. 숫자는 크게 6종류를 제공합니다.

```html
<!DOCTYPE html>
<html>
    <head> 
    </head>
    <body>
        
        <h1>Hello world</h1>
        <h2>Hello world</h2>
        <h3>Hello world</h3>
        <h4>Hello world</h4>
        <h5>Hello world</h5>
        <h6>Hello world</h6>

    </body>
</html>
```

## 인기테그
html은 기본적으로 6개의 제목 테그를 지원합니다. 하지만, 실제 개발에서 주로 사용되는 테그는 1~3 까지이 테그만 이용이 됩니다.
